🚁 Drone Project

A drone build using MicroPython that successfully responds to pitch, yaw, and roll inputs. This project involved integrating gyroscopes and accelerometers to read orientation data and translate it into motor control signals.

I implemented basic flight control logic that allows the drone to react to movement commands in real-time. While still in development for full flight capability, the system demonstrates functional sensor fusion and responsive control algorithms.

🎮 Hangman Game

A classic word-guessing game built in Java that challenges players to identify hidden words letter by letter. Players have limited attempts to guess correctly before the game ends, creating an engaging and strategic experience.

The game features an interactive interface that displays the current word progress, remaining attempts, and guessed letters. I implemented game logic to handle user input validation, win/loss conditions, and dynamic word display updates.
